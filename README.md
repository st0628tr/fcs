# FCS (Force Control System)

通信業界向けスタッフ・店舗・シフト管理システム

---

## 開発環境

- Laravel 12
- PHP 8.2
- MySQL
- TailwindCSS
- Vite

---

## 現在実装済み

### 認証

- ログイン機能
- 権限別ダッシュボード分岐

### マスタ管理

- 地域マスタ
- 店舗マスタ

### 店舗機能

- 店舗一覧
- 店舗追加
- 店舗編集
- 論理停止

### バリデーション

- 必須チェック
- 店舗コード重複チェック
- 半角英数字制限

---

## 今後実装予定

- スタッフ管理
- シフト管理
- CSV出力
- 実績管理
- パートナー管理
- 案件管理
